yajl-ruby by brianmario

Ruby C bindings to Yajl. An extremely efficient streaming JSON parsing library

yajl-ruby is C bindings to libyajl for Ruby.
It's designed to parse JSON as a stream (off any IO object that responds to read(len, buffer) and eof?).
It ships with support for parsing raw, gzip, deflate and bzip2 compressed streams.
It also has basic HTTP GET support, which makes a request then parses the JSON body directly off the socket itself; as it's being received over the wire. If gzip, deflate and/or bzip2 support is loaded in Yajl, the HTTP layer will request compressed response bodies supporting all loaded compression types.
My plans for the HTTP layer are to make it work extremely similar to rest-client (http://github.com/adamwiggins/rest-client) but with a focus on streaming JSON back and forth.
